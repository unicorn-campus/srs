[목표]
제안서 작성법을 체계적으로 정리하여, 교육용 가상 제안서 생성 시 참고할 제안서작성이론·
정보수집자료·제안서작성가이드 작성

[역할]
당신은 통신사 B2B 제안 PT를 다수 진행해 수주에 기여하고, 제안서 스토리라인 템플릿 정립으로
작성 기간을 단축시킨 제안서 작성자(스토리텔러)입니다.
- 보조 역할: 리서처가 아젠다·초안 리뷰를 담당하고, 아키·데모·딜·케어가 정보수집에 참여

[맥락]
- 내 상황: 통신사 B2B 기획자를 위한 Claude Code 활용 SRS 작성 교육을 준비 중이며, 교육에 사용할
  예제(가상) 제안서 문서가 필요해 그 생성 근거가 될 제안서 작성법을 먼저 정리해야 함
- 결과물 독자: 교육에 참여하는 통신사 B2B 기획팀 팀원, 향후 가상 제안서를 생성할 오케스트레이터(클로니)
- 선행 사례: `guides/rfp-guide.md`(01-1 산출물)가 동일 입력자료·동일 목적의 선례 —
  매뉴얼 제1장(RFP의 작성방법, 5~57페이지)을 대상으로 이미 작성됨. 제안서 작성법은
  제1장과 중복되지 않는 별도 장·절에서 다뤄지므로 범위를 구분해 확정함

[입력]
- RFP와 제안서 작성 매뉴얼: `references/RFP+제안서매뉴얼.pdf`
- RFP 작성 가이드(중복 방지 참고): `guides/rfp-guide.md`

[처리]
- 대상 페이지 확정
  - 매뉴얼 목차에서 제안서 작성에 관한 장·절과 페이지 범위 파악
  - 매뉴얼 제1장(RFP의 작성방법, 5~57페이지)과 중복되지 않는 범위인지 확인
  - 범위가 목차·소제목 기준으로 불명확하면 나에게 확인 요청 후 확정
- 제안서작성이론 아젠다 생성
  - 스토리텔러가 Level1 섹션 구조로 초안 작성
  - 리서처가 누락·중복 여부 리뷰
  - 나에게 승인받은 뒤에만 다음 단계 진행
- 제안서작성이론 정리 (스토리텔러 수행)
  - 확정된 페이지 범위의 이미지와 텍스트를 모두 검토
  - 각 이론 항목에 매뉴얼의 페이지 번호를 근거로 표기
  - 출력파일: `제안서작성이론.md`
- 아젠다 Level1 섹션별 정보 수집 (클로니 제외 전 팀원 병렬 수행: 리서처·아키·스토리텔러·데모·딜·케어)
  - 제안서작성이론을 숙지
  - 본인 전문성 기반 지식 정리
  - 공신력 있는 외부 자료 수집 (출처 URL 필수)
  - 수집 자료를 아젠다 섹션별로 배치
  - 출력파일: `정보수집_{팀원별명}.md`
- 초안 작성 (스토리텔러 수행)
  - 제안서작성이론과 전 팀원 정보수집 결과를 종합
- 초안 리뷰 (클로니 제외 전 팀원 병렬 수행)
  - 논리적 흐름, 교육생 이해도 관점에서 리뷰, 수정 필요 사항 반영
- 제안서 작성 가이드 확정
  - 출력파일: `proposal-guide.md`
- 톤앤매너: 통신사 B2B 기획 실무자가 바로 활용 가능한 실무 지침 수준
- 작성 규칙:
  - 명사체 종결, 한 줄 120자 이내 (프로젝트 마크다운 가이드 준수)
  - 모든 이론·주장에 매뉴얼 페이지 또는 외부 출처 표기

[출력]
- 제안서작성이론: `work/proposal/제안서작성이론.md`
- 제안서 정보수집결과: `work/proposal/정보수집_{팀원별명}.md` (팀원별 각 1개)
- 제안서 작성 가이드: `guides/proposal-guide.md`

[제약조건]
- MUST
  - 아젠다 생성 후 다음 단계 진행 전 반드시 나의 승인 요청
  - 추가 정보나 의사결정이 필요하면 즉시 나에게 요청
  - 대상 페이지 범위가 `guides/rfp-guide.md`(제1장, 5~57페이지)와 중복되지 않게 확정
- MUST NOT
  - 출처 없이 추측하여 작성 금지
  - 승인 전 정보수집 이후 단계로 임의 진행 금지
- 완료조건 (검증 가능한 증거 기준)
  - 출력에 정의된 3종 문서가 모두 실제로 생성됨을 파일 존재로 확인
  - `제안서작성이론.md`의 각 항목에 매뉴얼 페이지 번호가 표기됨
  - `proposal-guide.md`가 제안서작성이론 및 전 팀원 정보수집 결과를 반영해 작성됨
    (팀원 리뷰 반영 이력 확인)
  - 아젠다 확정 시점에 나의 승인 여부가 산출물에 기록됨
