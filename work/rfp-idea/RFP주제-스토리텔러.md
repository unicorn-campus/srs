# RFP 주제 — 스토리텔러 (고객 가치 관점)

> 관점: 최종 가입자 경험·MVNO 차별화 가치 중심 발상. 아이디에이션가이드 3절 MVNO 실수요(W1~W5)에 근거함.

---

## 후보1: SK망 알뜰폰 가입자 대상 실시간 사용량 통합조회 및 요금폭탄 방지 알림 서비스

- **발주 배경(WHY)**: W4(실시간 사용량 조회) 확장. SKT는 알뜰폰(MVNO)을 경쟁 상대로 인식해 KT·LG와  
  달리 통신사 차원의 통합 사용량 조회 플랫폼을 운영하지 않음. 이 때문에 SK망 기반 MVNO는 자체  
  조회 앱·홈페이지가 없으면 가입자가 사용량 조회 자체를 할 수 없음. 또한 종량제 요금제는 데이터  
  소진 후 자동 차단 기능이 없어 계속 과금되는 구조라 예상치 못한 "요금폭탄" 민원이 반복됨(As-Is)
- **업무 요구**
  - 가입자별 실시간 데이터·음성·SMS 잔여량 및 사용량 조회 (확인: 요청 후 3초 이내 응답, API 성공률 99.9%)
  - 데이터 소진 임계치(50%/80%/100%) 도달 시 실시간 푸시·문자 알림 발송 (확인: 임계치 도달 후 1분 이내 발송)
  - 종량제 요금제 소진 시 자동 차단 또는 소액 자동충전(스낵 데이터) 옵션 제공 (확인: 소진 시점부터  
    차단·충전 반영까지 5분 이내)
  - 사용량 조회 이력 및 알림 발송 이력 대시보드 제공 (확인: 최근 3개월 이력 조회 가능)
- **기술 요구**
  - MNO 과금·정책제어시스템(PCRF/PCF) 연동 REST API 제공 (지표: 조회 성공률 99.9% / 정상부하 / 월 측정)
  - 임계치 감지·알림 트리거 처리 성능 500 TPS 이상 (측정조건: 피크 시간대, 처리 지연 1분 이내)
  - OAuth 2.0 인증·전송구간 암호화(TLS 1.2 이상), MVNO별 접근권한 3단계 분리(관리자·운영자·조회자)
  - 임계치·알림 정책을 MVNO가 직접 설정할 수 있는 관리자 API·콘솔 제공
- **운용 요구**
  - 가용성 99.9% 이상 (연 다운타임 약 8시간 46분 이내)
  - 알림 발송 실패 시 30분 이내 재시도, 장애 통지 30분 이내, MTTR 4시간 이내
  - 월간 알림 발송 통계·API 호출 통계·오류율 리포트 제공
  - 신규 MVNO 온보딩 시 2주 이내 API 연동 완료 지원(기술 문서·샌드박스 환경 제공)
- **기대효과**: 요금폭탄·과금 관련 민원 월 300건 → 90건으로 70% 감소, 콜센터 사용량 문의 평균  
  응대시간 5분 → 1분 이내로 단축
- **시장 검증 근거**: SK망은 SKT가 알뜰폰 경쟁 구도상 통합 조회 플랫폼을 운영하지 않아 KT망("KT  
  마이알뜰폰")·LG망("LG 알뜰폰닷컴")과 달리 자체 사용량 조회 수단이 없는 소형 MVNO 가입자는  
  조회가 불가능함. 또한 종량제 요금제는 소진 후 자동 차단 기능이 없어 요금폭탄으로 이어짐이  
  확인됨. 이는 SK망 MVNO에 국한된 구조적 미해결 문제로, 제3자 앱만으로는 해결 불가능함  
  (근거: [알뜰폰 통신사별 사용량, 실시간 요금 조회하는 방법 - 모요](https://www.moyoplan.com/tips/%EC%9A%94%EA%B8%88%EC%A0%9C-%EC%A1%B0%ED%9A%8C/plan-real-time-watch),  
  [딸램 핸드폰 요금 폭탄 맞았습니다 - 클리앙](https://www.clien.net/service/board/park/18184963))

## 후보2: MVNO 가입자 대상 해외 로밍 실시간 사용량·요금 통합조회 서비스

- **발주 배경(WHY)**: W4(실시간 사용량 조회) 확장, 해외 로밍 특화. 로밍 요금은 해외 통신사업자의  
  TAP/CDR(과금상세내역) 정산 방식에 따라 확정되며, 자동로밍 이용 시 국가에 따라 최대 2~3개월  
  후에야 요금이 확정됨. 가입자는 로밍 이용 중 실시간 사용량·예상 요금을 전혀 확인할 수 없어  
  귀국 후 예상치 못한 "로밍 요금폭탄" 피해가 반복됨(As-Is). MVNO는 자체 해외 CDR 연동 인프라가  
  없어 MNO의 글로벌 로밍 정산망에 전적으로 의존해야 함
- **업무 요구**
  - 로밍 이용 중 준실시간 데이터·음성 사용량 및 예상 요금 조회 (확인: 로밍 CDR 수신 후 10분  
    이내 조회 화면 반영)
  - 로밍 요금 임계치(예: 1만원/3만원) 도달 시 실시간 알림 발송 (확인: 임계치 도달 인지 후 15분 이내 발송)
  - 로밍 요금제 미가입 상태의 해외 종량과금 이용 시 사전 안내 및 즉시 차단 옵션 제공 (확인: 출국  
    인지(로밍 등록) 시점 기준 안내 발송 성공률 99% 이상)
  - 귀국 후 로밍 사용 내역·확정 요금 조회 및 정산 대사 리포트 제공 (확인: 확정 요금과 청구서  
    금액 100% 일치)
- **기술 요구**
  - 해외 로밍 파트너망 TAP·준실시간 CDR 연동 및 가공 API 제공 (지표: CDR 수신-반영 지연  
    평균 30분 이내, 국가별 측정)
  - 준실시간 사용량 집계 처리 성능 200 TPS 이상(로밍 성수기 기준), 조회 응답 2초 이내
  - 로밍 국가·사업자별 요금 매핑 테이블 자동 갱신 체계, 요금 산정 오차율 1% 이내 정확도 보장
  - OAuth 2.0 인증, 해외 구간 전송 데이터 암호화(TLS 1.2 이상), MVNO별 국가·기간 단위 조회권한 통제
- **운용 요구**
  - 로밍 CDR 연동 가용성 99.5% 이상 (국제 구간 특성 반영)
  - 로밍 정산 오류·CDR 지연 장애 통지 1시간 이내, MTTR 8시간 이내(해외 파트너 의존 구간 고려)
  - 월간 국가별 로밍 CDR 수신 지연 현황 및 정산 대사 리포트 제공
  - 신규 로밍 파트너국 추가 시 연동 검증·오픈 리드타임 4주 이내 SLA 준수
- **기대효과**: 로밍 요금폭탄 관련 민원 분기 150건 → 40건으로 73% 감소, 로밍 요금 정산 분쟁  
  처리기간 평균 30일 → 7일로 단축
- **시장 검증 근거**: 로밍요금은 해외 통신사업자의 청구자료(CDR) 기준으로 부과되어 실시간 확정이  
  구조적으로 불가능하며, 자동로밍 이용 시 이용 국가에 따라 최대 2~3개월 후까지 청구될 수 있음이  
  확인됨. 현재 통신 3사가 제공하는 대응은 사전 로밍 가입·데이터 차단 설정 등 예방적 조치뿐이며,  
  이용 중 실시간 사용량·요금 확인 기능은 상용화되어 있지 않아 미해결 문제로 판단됨  
  (근거: [알뜰폰도 로밍이 되나요? - KB의 생각](https://kbthink.com/main/living-finance/mvno/2024/liivm-mvno4.html),  
  [알뜰폰도 로밍이 되나요? - 모요](https://www.moyoplan.com/tips/%EB%B6%80%EA%B0%80%EC%84%9C%EB%B9%84%EC%8A%A4/roaming))

  셀프개통 eSIM을 상용 제공 중이며 3분 내 개통이 가능함이 확인되어 배제함

  정책·행정 사유이며 기존 KTOA 신청서 절차로 해결 가능해 기술 시스템 구축형 RFP 소재로는  
  부적합하다고 판단해 배제함
