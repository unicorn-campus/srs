## 3. 통합 후보 상세

### C1. 개통 프로세스 실시간 심사·실패감지 및 본인확인 통합 시스템

- **제안자**: 리서처, 데모
- **병합 근거**: 리서처 후보1(안면인증·실명확인)과 데모 후보1(번호이동·개통 실패 감지·재처리)은  
  둘 다 "개통 신청 접수 후 처리가 중단되는 상황"(본인확인 실패 vs 사전동의·미납 등 정책 사유)을  
  실시간 판별·통지·재처리한다는 동일 업무 흐름(W3 실시간 프로비저닝)을 다루어 병합함
- **발주 배경(WHY)**: W3(실시간 프로비저닝 API) 확장 + 규제 준수. 2024년 대포폰 적발 97,399건 중  
  알뜰폰 비중이 92.3%(89,927건)에 달하나 안면인증·본인확인 엔진이 MNO 자체 채널 위주로 구축되어  
  MVNO 비대면 개통에는 실시간 연동이 없음. 또한 ARS 사전동의 미완료·요금 미납·번호이동 제한 등  
  정책적 사유로 개통이 중단돼도 사용자가 ARS·CS로 직접 재확인해야 하는 수작업 구조가 병존함(As-Is)
- **업무 요구**
  - 비대면 개통 신청 시 신분증 진위확인·안면인증(라이브니스 검증) 실시간 처리  
    (확인: 5초 이내 응답, 위변조 신분증 검출율 99% 이상)
  - 명의도용방지서비스(M-Safer) 연계 동일 명의 다회선 개통시도 조회·차단  
    (확인: 180일 내 4회선 초과 개통시도 자동 차단율 100%)
  - 개통·번호이동 신청 접수 시 실패 사유(본인확인 실패·사전동의 미완료·요금 미납·번호이동 제한 등)  
    통합 판별 (확인: 접수 후 5초 이내 실패 사유 코드 반환)
  - 실패 건 자동 재시도·고객 알림(SMS·앱푸시) 발송, 안면인증 실패 시 상담원 영상통화 등 대체 인증  
    자동 전환 (확인: 재시도·전환 개시 1분 이내)
  - 개통 진행상태(접수·본인확인·전산처리·완료) 및 거절·보류 이력·사유 실시간 대시보드 제공  
    (확인: 상태 변경 후 3초 이내 반영)
- **기술 요구**
  - MNO 안면인증엔진·번호이동/개통 시스템 통합 API 연동  
    (지표: 초당 200건 처리, 응답 5초 이내, 성공률 99.5% 이상 / 피크 시간대)
  - 실패 사유 코드 표준화 및 Webhook 기반 이벤트 통지 (지표: 이벤트 유실률 0.1% 이하, 월 측정)
  - 안면데이터·신분증 사본 전송구간 TLS 1.3, 저장 시 AES-256 암호화, 접근이력 로그 5년 보존,  
    ISMS 인증·CISO 지정 요건 충족
  - 이중화 구성을 통한 무중단 개통 처리 (지표: 장애 발생 시 자동 failover 30초 이내)
- **운용 요구**
  - 시스템 가용성 99.95% 이상 (연 다운타임 약 4시간 22분 이내)
  - 인증·개통 오류 통지 15분 이내, MTTR 2시간 이내
  - 전기통신사업법 시행령 등 법령 개정 시 정책 파라미터 무중단 반영, 월 1회 이상 규제준수·개통성공률  
    통계 리포트 제공
- **기대효과**: 알뜰폰 부정개통 비중 92.3% → 50% 이하로 축소, 개통 심사·처리시간 평균 1일 → 10분  
  이내 단축, 번호이동·개통 실패 관련 CS 문의 월 800건 → 480건(40% 감소)

### C2. 도매대가 사후규제 대응 협상·산정 지원 시스템

- **제안자**: 리서처, 딜
- **병합 근거**: 리서처 후보2(도매제공의무 사후규제 대응 산정·협상 관리)와 딜 후보1(코스트플러스  
  원가검증 시뮬레이션)은 둘 다 2025년 사후규제 전환·60일 협상기한·코스트플러스 방식 도입이라는  
  동일 정책 배경(W1)에 근거하며, 산정 근거자료 관리와 원가 시뮬레이션이 상호 보완적이어서 병합함
- **발주 배경(WHY)**: W1(망 도매 정산 자동화) 확장 + 규제 준수(전기통신사업법 제38조의2). 2025년  
  시행령 개정으로 도매대가 협상이 사전규제에서 사후규제로 전환되며 기한이 90일에서 60일로  
  단축되고, 리테일마이너스 방식에 코스트플러스 방식이 추가돼 산정 근거자료 요구가 복잡해짐.  
  MNO가 망 원가를 선제 공개해야만 검증 가능한 구조라 MVNO는 자체 검증 수단 없이 MNO 제시안을  
  그대로 수용하거나 근거 부족으로 기한을 넘기는 리스크에 노출됨(As-Is)
- **업무 요구**
  - 리테일마이너스·코스트플러스 방식별 산정 근거자료(원가·회피가능비용 내역) 자동 산출·제공  
    (확인: 요청 후 3영업일 이내 제공)
  - 요금제·가입자 세그먼트별 도매대가 방식 손익 시뮬레이션 및 코스트플러스 원가 구성요소(설비상각비·  
    운영비·이윤율 등)별 적정성 검토 리포트 생성  
    (확인: 표본 요금제 20종 시뮬레이션 결과 오차 5% 이내, 원가 항목 10종 이상 반영 리포트 10분 이내 산출)
  - 협상 진행 단계(요청-자료제공-협의-체결) 마일스톤 관리 및 60일 기한 임박 자동 알림  
    (확인: 기한 도래 D-10/D-3 자동 알림 발송율 100%)
  - 금지행위(부당 정보요구, 산정주기 일방 변경 등) 의심 이력 자동 탐지·월간 리포트 생성  
    (확인: 협정 위반 의심 건 월 1회 이상 자동 리포트 산출)
  - 도매대가 변경 이력·협정 문서 버전관리 및 MVNO 열람 포털 제공 (확인: 열람 요청 후 1일 이내 응답)
- **기술 요구**
  - 정산·원가 데이터 배치+실시간 하이브리드 연동 (지표: 일 배치 정산 데이터 매일 06시까지 100% 적재)
  - 요금제 100종 동시 손익 시뮬레이션 처리 10분 이내(협상 준비 피크 기간 기준), 외부(공시자료·  
    통계청 등) 원가 참조 데이터 연계 API (지표: 갱신 주기 월 1회 이상, 연계 성공률 99% 이상)
  - 산정 근거자료·시뮬레이션 이력 위변조 방지 전자서명·해시 검증, 접근권한 RBAC 3단계(관리자·  
    협상담당자·조회자), 저장 데이터 암호화·전송구간 TLS 1.2 이상
  - 규제기관 자료제출 요청 대응 API 제공, 감사로그 3년 이상 변경불가 저장구조로 보존, 시스템 장애  
    시 협상 이력·시뮬레이션 데이터 일 1회 자동 백업(복구시간 4시간 이내)
- **운용 요구**
  - 시스템 가용률 99.9% 이상 (연 다운타임 약 8시간 46분 이내)
  - 정산 데이터 대사 불일치 발견 시 4시간 이내 1차 소명자료 제공(MTTR), 협상 기한 미준수·분쟁  
    발생 시 에스컬레이션 절차 가동 및 법무 검토 요청 접수 후 1영업일 이내 착수
  - 원가 참조 데이터·시뮬레이션 로직 변경 시 사전 통지(최소 7일 전) 및 변경 이력 관리, 협상 담당자  
    대상 초기 도입 교육 1회 이상 실시 및 사용자 매뉴얼 제공
  - 협상 이력·산정 근거 자료 5년 이상 보관(분쟁 발생 시 사후규제 대응 근거 확보)
- **기대효과**: 도매대가 협상 준비 기간 평균 30일 → 10일로 단축, 전체 협상 소요기간 90일 → 45일  
  이내 단축, 60일 법정 기한 내 협상 완료율 60% → 95%로 개선, 월간 정산 분쟁 건수 15건 → 3건  
  이하로 축소

### C3. 도매 용량·QoS 실시간 가시성 대시보드

- **제안자**: 아키, 데모
- **병합 근거**: 아키 후보2와 데모 후보2는 둘 다 MVNO가 도매 대역의 실시간 용량·QoS를 확인할  
  수단이 없어 협상·품질 대응이 사후적이라는 동일 문제(W5)를 다루어 병합함. 아키는 데이터  
  파이프라인·저장구조, 데모는 혼잡지역 자동 리포트·연동 표준(NETCONF)을 보완적으로 제공함
- **발주 배경(WHY)**: W5(도매 용량·트래픽 가시성) 근거. 알뜰폰은 도매 대역(특정 용량)만 임대해  
  트래픽 집중 시 품질이 저하될 수 있으나 MVNO는 실시간 사용률·QoS·혼잡도를 확인할 방법이 없어  
  품질저하를 가입자 민원 접수 후에야 사후적으로 인지함. 2022년 도매제공 의무제 일몰에 이어  
  2026년 4월부터 협상 기한도 90일에서 60일로 단축되는 사후규제로 전환되어 정량 근거자료  
  확보가 더욱 중요해짐(As-Is)
- **업무 요구**
  - 도매 대역별·지역별·시간대별 실시간 트래픽 사용률·혼잡도 조회 (확인: 요청 후 3초 이내 응답,  
    5분 단위 갱신)
  - 가입자 체감 품질지표(지연·손실률·처리속도) 추이 대시보드 제공 및 이상치 자동 하이라이트  
    (확인: 최근 30일 이력 조회 가능)
  - 용량 임계치(예: 사용률 80%) 또는 품질 임계치 초과 시 사전 경보 알림  
    (확인: 임계치 도달 후 5분 이내 알림 발송)
  - 용량 협의용 리포트 자동 생성(피크타임 혼잡 지역 상위 10곳) (확인: 일 1회 자동 생성, 수기  
    집계 대비 오차 0%)
- **기술 요구**
  - MNO 네트워크 장비(NMS·TMS)로부터 표준 API(NETCONF·REST) 기반 실시간 트래픽 데이터 연동  
    (지표: 수집 지연 1분~5분 이내, 연동 성공률 99.5% 이상)
  - 대시보드·API 조회 성능 200 TPS 이상(피크 시간대, 응답 3초 이내), 대용량 시계열 데이터 저장소  
    최소 13개월 보관(정량 협상 근거자료용)
  - API 인증 OAuth 2.0 및 TLS 1.2 이상 암호화, 사업자별 데이터 접근 격리(멀티테넌시)  
    (지표: 타 MVNO 데이터 접근 차단율 100%)
- **운용 요구**
  - 대시보드·API 가용성 99.9% 이상 (연 다운타임 약 8시간 46분 이내)

- **기술 요구**
  - Webhook/API 기반 실시간 이벤트 푸시 (지표: 발생~통지 지연 1분 이내 / 성공률 99.9% / 월 측정)
  - 대규모 동시 조회 처리 300 TPS 이상 (측정조건: 장애 피크시간대, 응답 2초 이내)
  - 다중 채널 통지(API+이메일+SMS) 이중화, 단일 채널 장애 시 자동 전환 (지표: 전환 30초 이내)
  - 접근 인증 OAuth 2.0, 전송구간 암호화(TLS 1.2 이상), 통지 이력 위변조 방지 로그 보관
- **운용 요구**
  - 통지 시스템 자체 가용성 99.95% 이상 (연 다운타임 약 4시간 22분 이내)
  - 장애 통지 SLA 준수율 월 98% 이상, 미준수 시 페널티 조항 반영
  - 24x365 공동 대응 핫라인 운영 및 분기 1회 합동 장애대응 훈련(모의훈련) 실시
  - 월간 장애대응 리포트(통지 소요시간·MTTR·재발률) 제공
- **기대효과**: 장애 인지~통지 소요시간 평균 40분 → 5분 이내 단축, 장애 발생 시 MVNO 고객센터  
  문의량 30% 감소(예: 월 5,000건 → 3,500건)

### C9. MVNO 고객센터용 가입자별 서비스 장애 실시간 진단 포털

- **제안자**: 케어
- **발주 배경(WHY)**: W3(실시간 프로비저닝)·W4(실시간 사용량 조회)를 사후 진단 영역으로 확장한  
  실수요임. MVNO는 자체 망·NOC이 없어 유심 불량·개통 지연·품질저하 등 개별 가입자 이슈가  
  접수돼도 MNO 측에 문의해야만 원인을 확인할 수 있음(As-Is). 소비자단체 조사에 따르면 알뜰폰  
  시장 확대 과정에서 AS 인프라 미비로 "고객센터 불통, 개통 지연, 품질 불량"이 최다 민원 유형으로  
  나타났으며, 원인 진단 도구 부재가 상담 장기화·재문의를 유발함
- **업무 요구**
  - 가입자 번호 기준 회선상태(활성/정지/장애)·유심 상태 실시간 조회 (확인: 3초 이내 응답)
  - 최근 24시간 내 개통·인증·과금 오류 이력 조회 (확인: 표본 100건 대사 결과 100% 일치)
  - 기지국·권역 단위 품질저하(호절단률·데이터속도 저하) 여부 조회 및 알림 (확인: 임계치 초과  
    시 5분 이내 CS 담당자 알림)
  - 진단 결과 기반 1차 조치 가이드(표준 응대문구) 자동 제공
- **기술 요구**
  - MNO OSS/NOC 연동 조회 API 응답시간 2초 이내 (지표: 성공률 99.9% / 정상부하 / 월 측정)
  - 동시 조회 처리 500 TPS 이상 (측정조건: 전체 CS 상담원 동시 접속 피크시간대)
  - 상담원 권한별 3단계 접근제어(관리자·상담원·조회자), 조회 이력 감사로그 1년 보관
  - 야간·공휴일 포함 24x365 실시간 API 제공(배치 처리 미의존)
- **운용 요구**
  - 포털 가용성 99.9% 이상 (연 다운타임 약 8시간 46분 이내)
  - 장애 통지 30분 이내, MTTR 4시간 이내
  - 신규·기존 CS 상담원 대상 분기 1회 교육 및 진단 매뉴얼 갱신
  - 월간 이용 통계(조회건수·1차 해결률) 리포트 제공, 1차 해결률 목표 70% 이상
- **기대효과**: CS 1건당 평균 처리시간 12분 → 6분 단축, 동일 건 재문의(콜백) 비율 25% → 10%로 감소
