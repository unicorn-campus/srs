## 스토리텔러 리뷰

> 대상: `work/srs/srs-draft.md`(개발요구사항정의서 16장 1차안) / 기준: `guides/srs-guide.md`(G0~G16)  
> 관점: 참조 양식(LG유플러스 SmartQA) 충실도·가독성·중복·명료성. 심각도순 나열.

### 총평(구성 충실도 — 이상 없음)

- [구성] 전체: 머리말(제목·교육용 안내·문서정보) → 1~16장 → 꼬리말(교육용 고지) 골격 완비 → 준수, 유지.  
- [구성] 표 4종(9.1 룰 적용단위·11장 상태값·14장 화면·15장 성능) + ⚠ 주의문(4장·13장) + 수행/미수행  
  경계(3.1/3.2) + X-RAY 예외(16장) 모두 재현 → 준수, 유지.  
- [구성] 방법론 부록(승인 상태·리뷰 반영 내역·RTM) 미포함 확인 → 참조 양식 준수(제거 지적 없음). 9.3·13.1의  
  "승인자"는 룰 등록·감사 필드로 정당한 본문이며 문서 메타 부록이 아님.

### 지적 목록(심각도순)

- [모호] 10.3 ↔ 11장: 10.3 지연 처리 상태(지연 대기/진행 중/완료/실패)와 11장 결과 상태값(통지 지연·조회  
  지연 등)이 모두 "지연" 명칭을 쓰나 서로 다른 축(처리 진척 vs 결과 판정)임에도 관계 설명이 없어 개발자가  
  동일 상태 집합으로 오해할 소지 → 10.3 서두에 "11장 결과 상태와 별개인 지연 큐 진척 상태임"을 1문장 명시.  
- [중복] 7.3 ↔ 8장 ↔ 10.2: "대량 통지/클레임 구간은 일부 대상 우선 통지" 규칙이 세 장에 반복 서술됨  
  → 정본을 10.2(전환 규칙)로 두고, 7.3·8장은 "우선 통지 전환은 10.2"로 상호참조만 남겨 중복 축소.  
- [중복] 6장 ↔ 15장: 성능 수치(99.9%·30초·P99 3초·3초·10분·15분·3영업일)가 6.2~6.7 임계값과 15장 표에  
  이원 기재되어 정본이 모호하고 향후 수치 불일치 위험 → 15장 성능표를 정량 정본으로 선언하고, 6장은 판정  
  논리 서술 + "정량 기준은 15장" 참조로 통일.  
- [모호] 12장/15장 요구ID: "인지 ~ 담당자 5분 종단" 지표의 근거 요구ID가 15장 [REQ-BIZ-01]과 12장  
  [REQ-OPS-02]로 갈림 → 종단 SLA 근거 요구ID를 한쪽으로 통일(스펙 확인 필요). 독자 추적성 혼선 방지.  
- [개선] 6.2 ↔ 16장: 부분집합 기호 방향이 6.2 "5분 ⊃ 1분"과 16장 "1분 ⊂ 5분"으로 혼용됨(논리는 동일)  
  → 문서 전체를 "1분 ⊂ 5분" 한 방향으로 통일하여 오독 방지.  
- [중복] 8장 ↔ 10.1: "신규 사고유형(전체/팀/담당자 최초 발생)" 정의가 두 장에 실질 중복 나열됨(상호참조로  
  일부 완화됨) → 정의 정본을 10.1로 두고 8장은 "신규 트리거 정의는 10.1" 참조로 대체.  
- [중복] 6.2 ↔ 7.2 ↔ 8장 ↔ 15장: "채널 자동 전환 30초"가 4개 장에 반복 → 정량 정본을 15장으로 두고  
  나머지는 조건 서술만 유지, "(30초, 15장)" 형태로 경량 참조.  
- [개선] 5장 서두: 소개 문단이 "검증·정규화·AI 입력 전처리"만 열거하고 5.4(캐시 신선도·대사 기준)를  
  누락 → 서두에 "(5.1 ~ 5.4)" 범위와 5.4를 포함하도록 보완(4·6·7장은 범위 표기됨, 5장만 누락).  
- [개선] 11장: 통지 결과 상태(정상/부분/실패)와 SLA 위반 유형(통지 지연·조회 지연·응대 지연·리포트 지연·  
  가용성 미달)이 한 표에 혼재되어 표가 길어짐 → 유지 가능하나, 상태 열에 "결과/SLA위반" 구분 태그를 병기하면  
  운영자·개발자 판독성 향상.  
- [개선] 면책 규칙 분산: 면책이 3.2·6.6·8장·9.4·11장·13.1·16장 7개 장에 흩어져 전모 파악이 어려움  
  (관점별 필요 반복이나) → 16장 면책 케이스에 "면책 근거 로그 13.2·준수율 분모 9.4" 상호참조를 덧붙여  
  독자가 면책 전체 흐름을 한 진입점에서 추적 가능하게 보완.

### 명사체·용어 일관성

- [개선] 명사체(~함/~임) 종결 전반 준수 → 유지. ETR·HITL·MNO·MVNO 표기 일관 → 유지.  
- [개선] "수신 ~ 통지 1분"·"인지 ~ 담당자 5분" 수식어가 "(시스템 전송)/(전송 구간)", "(운영 포함 종단)/  
  (종단)"으로 장마다 미세하게 다름 → 용어집 1행으로 표준 표기를 고정하여 12·15·16장 표현을 통일 권장.
